Cette page web pollue !

Il existe de nombreuses habitudes de web-design qui peuvent rendre vos pages web moins polluantes. Nous allons vour quels sont ces habitudes.


Pourquoi le web pollue ?



A cause du stockage !

Le code d'un site web doit être stocké dans un data center. Les data-centers représentent la majorité de la pollution émise par le web car ils consomment beaucoup d'énergie.

Aujourd'hui avec l'explosion du nombre de sites web il est nécessaire d'adopter une démarche de développement web écoresponsable.

En diminuant la taille de votre code source vous diminuez le coût de stockage de votre page web et donc son coût en énergie.

Comment faire pour alléger sa page web ?

• Minimiser le code source.

• Minimiser les scripts

• Minimiser le design do votre page et les effets superflu

• Minimiser les balises "div" et "class" superflu

• Utiliser des langages de programmation économes

• Minimiser les échanges d'informations et les requêtes entre utilisateur et serveur

La clé c'est l'optimisation et le minimalisme !

Un site web de 100 lignes de code peut donner à l'utilisateur les mêmes informations qu'un site web de 1000 lignes de code !



Le contenu de la page

Pourquoi faut-il limiter les médias sur votre page ?


Quand un utilisateur se connecte sur votre page web une requête http envoie sur son navigateur tous les fichiers dont il a besoin pour l'affichage. Ces rêquete consomment de l'énergie.

De plus chaque média que vous hébergez vous-même a un coût de stockage dans les data centers.

Comment faire pour bien gérer les médias ?


• Limiter au maximum le nombre de médias sur votre page, garder uniquement les médias essentiels.

• Compressez et réduisez la taille de vos images. Adaptez leur résolution à votre utilisation.

• Éviter les images en haute résolution où compressez-les.

• Privilégier aussi les images vectorielles et les icônes.

• Si le média est déjà hebergée ailleurs, utiliser un lien plutôt que de la réhéberger.

• Diminuer la résolution de vos vidéos au minimum. Et éviter l'auto play.


En plus de rendre votre site plus écologique, diminuer le nombre de média va aussi le rendre plus rapide !



Autres conseils



Les polices d'écritures

Il existe plusieurs types de polices d'écriture.

• Il y a les polices natives : celles-ci sont déjà installés sur vos appareils :

Exemples de polices natives :

Arial
Times New Roman
Garamond
Courier New
Verdana

• Et il y a les polices customisées :

si vous décidez de les utiliser sur votre page le serveur aura besoin d'envoyer des requêtes pour afficher et télecharger ces polices.

Si vous minimisez le nombre de rêquetes votre sit sera mieux optimisé et consommera donc moins d'énergie.
En écoconception il est d'ailleurs recommandé de s'en tenir à deux polices d'écriture maximum.

Un design simple

• Plus le design de votre page web est simple, plus votre page sera écologique. Car cela limitera les échanges d'informations et les rêquetes

• Minimisez le nombre de boutons,évitez de mettre trop de dynamisme. Privilegiez une page statique et donc économe.

• Assureé vous que l'utilisateur trouve rapidement les informations qu'il cherche et ne soit pas perdu (chaque clic est un échange d'informations qui coute de l'énergie)

• Choisissez soigneusement vos couleurs. Certaines couleurs comme le bleu consomment plus d'énergie. En écoconception il est recommandée de s'en tenir à trois couleurs maximum.

• Minimisez le nombre d'animations (exemple : animations Javascript).
